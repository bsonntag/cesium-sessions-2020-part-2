Na sessão anterior fizemos um recap das bases de JavaScript e vimos também algumas partes mais avançadas da linguagem.
Hoje vamos começar por rever como se interage com o conteúdo de uma página web a partir de JavaScript. Vou mostrar-vos como usar apenas JavaScript para aplicações complexas se torna complicado, e como frameworks como React surgiram para simplificar esse trabalho.
Depois vamos então ver as bases de React, e como React nos permite simplicar a tarefa de interagir com a DOM.
Nas duas últimas sessões vamos falar de tópicos mais avançados de React, e vamos ver algumas bibliotecas e frameworks construídas por cima de React.
